# Week 14: Interview Prep

## Resume Tips
- Lead with "AI Infrastructure Engineer" or "AI Platform Engineer"
- Highlight: AWS + Kubernetes + Networking + AI/ML
- Quantify: "Deployed LLM serving 10K requests/sec at P99 < 200ms"
- Portfolio project link prominently

## LinkedIn Keywords (Add these)
- AI Infrastructure
- ML Platform
- GPU Computing
- LLM Deployment
- vLLM / Triton
- MLOps
- SageMaker
- Bedrock
- RAG
- Kubernetes for ML
- Distributed Training

## System Design Questions (Common)
1. Design an LLM serving platform for 100K users
2. Design a RAG system for enterprise documents
3. Design ML training infrastructure for a team of 50 data scientists
4. Design cost-efficient inference for multiple models
5. Design a multi-tenant AI platform

## Behavioral Questions
- Tell me about a time you scaled infrastructure
- How do you handle trade-offs between cost and performance?
- Describe a complex networking problem you solved

## Target Companies (Apply in this order)
1. Product companies (Oracle, Guidewire, BP) — easier to get
2. GCCs (Google, NVIDIA, Amazon) — higher pay
3. AI Startups (Sarvam, etc.) — exciting work + ESOPs
4. Consulting (Deloitte AI practice) — good exposure
